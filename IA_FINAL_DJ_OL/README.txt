ProjetIA
jeu du taquin 
Denou Julien et Ludivine Obry
code en python 2.7.13 sous Eclipse Luna (PyDev)
Une version plus récente vous demandera de changer le Tkinter en tkinter
Le fichier python est present dans le dossier avec le package utilise.
Le tableau excel presentant les resultats des 50 essais du programmes entre les differentes heuristiques selon leur temps de calculs uniquement est present dans ce dossier au nom de Tableau_Resultats_Heuristiques.


Notes de version :
Version du logicielle fonctionnelle cependant de nombreux points peuvent etre ameliores ou revu tel que :
L'ajout d'un tableau recapitulatif des resultats de chaque heuristiques
La mise en place d'une creation aleatoire du taquin
L'ajout d'image a la place de chiffre
La possibilite de faire des taquins plus grand
Une sortie enregistrant automatiquement les resultats
